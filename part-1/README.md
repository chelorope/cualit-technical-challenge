# Part 1 - CRUD application

Full-stack CRUD Application

## Install dependencies

```shell
npm install
```

## Database

- Create a new DB

- Update DB config inside `backend/config`

Default DB config:
```json
{
  "host": "127.0.0.1",
  "port": "3306",
  "username": "root",
  "password": "",
  "database": "cualit",
  "dialect": "mysql",
  "logging": false
}
```

## Run project

```shell
npm start
```
